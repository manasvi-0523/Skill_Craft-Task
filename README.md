# Skill_Craft-Task

SkillCraft Technology Cybersecurity Internship projects. Four tasks, one repository.

---

## Tasks

| Folder | Task | Stack |
|--------|------|-------|
| `SCT_CS_01` | Caesar cipher tool with brute-force and frequency analysis | Python, Streamlit |
| `SCT_CS_02` | Password strength checker (CLI + web) | Python, HTML/JS |
| `SCT_CS_03` | Typing behavior research tool | React, Vite |
| `SCT_CS_04` | Image encryption tool (XOR + RGB shift + pixel shuffle) | Python, Tkinter |

Each folder has its own README with setup instructions, usage examples, and test results.

---

## Quick start

**Python tasks (01, 02, 04)**

```bash
git clone https://github.com/manasvi-0523/Skill_Craft-Task.git
cd Skill_Craft-Task

pip install -r SCT_CS_01/requirements.txt
streamlit run SCT_CS_01/app.py

python SCT_CS_02/password_strength_checker.py
# or open SCT_CS_02/index.html in a browser

pip install -r SCT_CS_04/requirements.txt
python SCT_CS_04/app.py
```

**React task (03)**

```bash
cd SCT_CS_03
npm install
npm run dev
```
